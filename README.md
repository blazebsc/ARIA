# ARIA

A Minecraft mod featuring an AI companion entity that can speak, listen, and interact with players. ARIA evolves over time through stages, with changing behaviors and visual states.

## Features

- Voice-based interaction (speech recognition & text-to-speech)
- Dynamic face states and expressions
- Progressive stages that evolve over time
- Companion AI that follows and questions players
